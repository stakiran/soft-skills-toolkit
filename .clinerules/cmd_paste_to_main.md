# Paste To Main スクリプト

## 背景
- エントランスページとは、入口を担うファイルを指す
    - ここでは README.md と Index.md を指す
- エントランスページ xxxx.md のつくりかたは次のとおり:
    - 1: xxxx_JP.md をマスターとして執筆する。日本語で執筆している。
    - 2: translator を使って、xxxx_JP.md から英語版 xxxx_JP_EN.md に変換する
    - 3: xxxx_JP_EN.md の内容を xxxx.md にコピペする

つまりエントランスページは英語版としてつくるが、筆者が日本語ネイティブであり翻訳をはさんでいるため、日本語原稿 → 中間生成物（英語） → 最終成果物が分かれている。またコピペが必要になっている。

## やりたいこと
手順 3: のコピペを自動化したい。

## 指示
エントランスページの英語版 xxxx_JP_EN.md の内容を、エントランスページ xxxx.md に書き込む python スクリプトをつくれ。

`python paste_to_main.py` を実行することで動かす。

stdout は不要。
