この文書では基礎用語の定義と解説を書く。

# Soft Skills Toolkit(sstk)
ソフトスキルを利活用しやすくするためのツール。

## 性質
- ハード・ツールではなくソフト・ツールを指す
- 1-ツール 1-Markdown で提供する

## 提供方法
- GitHub および静的サイト GitHub Pages にて提供する
- ツールの一覧（カタログ）は index.md にて提供する

# ツール
単にツールと書いたとき、ソフト・ツールを指すことが多い。

# ハード・ツール
ハードウェアやソフトウェアなど、具体的に実体があって使える道具。

# ソフト・ツール
テンプレートやチェックリストや手順書など、概念的な道具。

本文脈で単に「ツール」と書いたときは、ソフト・ツールを指すことが多い。
