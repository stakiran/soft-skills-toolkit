## 解説
- ツールとはソフトウェアではなく、テンプレートやチェックリストや手順といったものを指します
- Markdown で書かれており、読者はコピペベースですぐに利用できます
    - もちろん AI エージェントとの連携も可能です
- 対応言語は以下のとおり
    - 日本語
    - 英語（ファイル名末尾に `_EN`）

## ツールの開発
- ツールの作成
    - Markdown で書きます。テンプレートは _new.md です
- ツールの翻訳
    - translator.py を使います。VSCode 上でタスクとしてすぐに呼び出せます
    - OpenAI API を使用しています
- ツールのデプロイ
    - 今のところ、GitHub を直接見てもらう想定です
    - ★静的サイトも検討したい。GitHub Pages で十分とは思う

## エントランスページ

## Cline との付き合い方
- スクリプトをつくるときに利用します:
    - .clinerules/ 配下に仕様を書き、Cline に「～～をつくって」とお願いすることでつくります
    - SDD には則っておらず、.clinerules/ 配下はインプットにすぎません

## リポジトリの運用
- ブランチは master 一本です
    - sstk は「常に未完成」であるため
    - 最新に問題がある場合、適当に遡ってください
- Issue は歓迎します
- Pull Request は受け付けていません
