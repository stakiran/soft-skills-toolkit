# Soft Skills Toolkit

## ここは？
ようこそ。私はソフトスキル・エンジニアリングを立ち上げており、理論と概念については [sta - DEV Community](https://dev.to/stakiran) にて扱っています。ここはソフトスキルの発揮に役立つ **ツール** を公開する場所です。

## 使い方
- ツールとはソフトウェアではなく、テンプレートやチェックリストや手順といったものを指します
- Markdown で書かれており、読者はコピペベースですぐに利用できます
    - もちろん AI エージェントとの連携も可能です
- 対応言語は以下のとおり
    - 日本語
    - 英語（ファイル名末尾に `_EN`）
