### 解説
- ツールとはソフトウェアではなく、テンプレートやチェックリストや手順といったものを指します
- Markdown で書かれており、読者はコピペベースですぐに利用できます
    - もちろん AI エージェントとの連携も可能です
- 対応言語は以下のとおり
    - 日本語
    - 英語（ファイル名末尾に `_EN`）

## 開発者向け: 開発について
ツールについて:

- ツールの作成
    - Markdown で書きます。テンプレートは _new.md です
- ツールの翻訳
    - オリジナルは日本語です
    - 英語への翻訳は translator.py を使います。VSCode 上でタスクとしてすぐに呼び出せます
    - 翻訳は OpenAI API を使用しています
- ツールのデプロイ
    - 今のところ、GitHub を直接見てもらう想定です
    - ★静的サイトも検討したい。GitHub Pages で十分とは思う

リポジトリの運用について:

- ブランチは master 一本です
    - sstk は「常に未完成」であるため
    - 最新に問題がある場合、適当に遡ってください
- Issue は歓迎します
- Pull Request は受け付けていません
    - この sstk は私の作品だからです
    - 「みんなでつくる sstk」用のリポジトリは、そのうち別途つくりたい
        - awesome-sstk とか？
