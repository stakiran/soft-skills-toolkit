# Soft Skills Toolkit

## What's this?
Welcome. I'm building "Soft Skills Engineering," and I cover the theory and concepts on [sta - DEV Community](https://dev.to/stakiran). This is where I publish **tools** that help you apply soft skills in practice.

## How to use
- "Tools" here don't mean software; they refer to things like templates, checklists, and procedures
- Everything is written in Markdown, so you can use it immediately via copy-and-paste
    - You can, of course, also integrate it with AI agents
- Supported languages are as follows
    - Japanese
    - English (files with the `_EN` suffix)
