# Personal Daily Workspace(PDW)
- 自分用のテキストベースの作業領域を日単位で確保すること
- 1日の運用:
    - 1: 今日の領域をつくる（昨日の分からコピペ）
    - 2: 作業をしつつ必要に応じてメモ
    - 3: 今日の終わりに軽く振り返りや整理を行う
- メリットはテキストとして残ること。振り返りや生成 AI 連携がしやすい

# Tools
PDW に書き込むと捗るかもしれないもののリスト:

```markdown
- タスクリスト、特に今日やること・明日やること・今週やること・それ以降の四つに分ける
- 連絡待ちリスト: `(送信日) (誰に) (用件)`
- YWT(やったこと、わかったこと、次にやること)
- 思いついたアイデア
- 一息には書けない文章の下書き
- ポジティブな日記
- ネガティブな日記（残り続けるとストレスになるため翌日以降は消去推奨）
- もやもやした考え事や悩み事の言語化
```

どこで PDW するかを検討するのに使えるトリガーリスト:

```markdown
- ローカルでファイルに書く？クラウドツールを使う？ローカルで書いてクラウドにアップする？
- プレーンテキスト？リッチテキスト？
- Markdown？フリーフォーマット？
- 画像は扱う？テキストだけでいい？
- ファイル？ページ？スライド？
- チャット、ウィキ、ノート、Issues などチケット、ホワイトボードで実現できそう？
- 1日1ページにする？1週間1ページにする？1ヶ月1ページにする？
- 毎日その日の枠をつくるのは手作業？自動化する？
- 生成 AI とはどのように連携する？またプロンプトは？
```
