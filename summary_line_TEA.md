# サマリーを書くための TEA ライン
- サマリーのフォーマット
- 概要と効果を端的に記せる
- TEA:
    - Target: 誰に対してのものか
    - Effect: Target に及ぼす効果（できれば数字で）
    - Action: Target に Effect を及ぼすために具体的に何をするか・どう捉えるか
- それぞれ一行で書く

# Tools
TEA ラインのテンプレート

```markdown
# TEA
- Target: (誰に対してのものか)
- Effect: (Targetに及ぼす効果を数字で)
- Action: (そのために具体的に何をするか・どう捉えるか)
```

TEA を使うタイミングを問うチェックリスト:

```markdown
全部 No の場合に、TEA を使っても良い。

- tl;dr を書いてもいい場面か？書いたとして通じるか？
    - y | n
- エグゼクティブサマリーを書くべき場面か？
    - y | n
- サマリーの項を設けて、1ページ程度の情報量を書かねばならないか？
    - y | n
```

# 補足: TEA の意図
- Target は Who を指定する
- Effect は Why を指定する
- Action は How または What を指定する
